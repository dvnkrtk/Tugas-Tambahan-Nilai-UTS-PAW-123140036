# 05: Unit Tests and pytest

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami pengujian otomatis (automated testing) pada aplikasi Pyramid menggunakan pytest. Dengan pengujian ini, developer dapat memastikan bahwa setiap fungsi, view, dan route berjalan sesuai harapan tanpa harus menguji secara manual melalui browser.

## Analisa
Unit Testing adalah proses menguji potongan kecil kode (unit) secara terpisah, biasanya berupa fungsi atau method, untuk memastikan perilakunya benar. Pada  guide ini diberitahu cara menginstall pytest agar dapat melakukan pengetesan secara otomatis.
