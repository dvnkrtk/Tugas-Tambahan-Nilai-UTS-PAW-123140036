# 20: Logins with Authentication

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami cara membuat sistem login pada aplikasi web Pyramid yang aman dengan mekanisme autentikasi. Sistem autentikasi digunakan untuk memverifikasi identitas pengguna sebelum memberikan akses ke fitur atau data tertentu dalam aplikasi.

## Analisa
Berdasarkan hasil percobaan, implementasi login dengan autentikasi memungkinkan aplikasi membedakan pengguna yang sah dan tidak sah. Pyramid menyediakan cara mudah untuk mengelola session dan autentikasi, sehingga informasi login dapat disimpan sementara di sisi server.  
Penggunaan autentikasi membantu menjaga keamanan data, mencegah akses tidak sah, dan mempermudah pengelolaan hak akses pengguna. Secara keseluruhan, modul ini menekankan pentingnya autentikasi dalam aplikasi web untuk memastikan pengalaman pengguna yang aman dan terkontrol.
