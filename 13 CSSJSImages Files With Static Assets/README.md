# 13: CSS/JS/Images Files With Static Assets

## Tujuan Percobaan
Percobaan ini bertujuan untuk mempelajari cara menangani dan melayani file statis (CSS, JavaScript, dan gambar) dalam framework Pyramid. File statis ini digunakan untuk mempercantik tampilan halaman web serta menambah interaktivitas antarmuka pengguna.

## Analisis
Berdasarkan hasil percobaan, saya memahami bahwa Pyramid menyediakan cara khusus untuk menangani file statis menggunakan perintah `config.add_static_view()`. Fungsi ini digunakan untuk menentukan rute (URL) di mana file statis dapat diakses oleh browser.
