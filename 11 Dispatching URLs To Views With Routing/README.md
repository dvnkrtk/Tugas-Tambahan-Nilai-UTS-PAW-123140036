# 11: Dispatching URLs To Views With Routing

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami bagaimana routing bekerja dalam framework Pyramid, yaitu proses yang menghubungkan URL tertentu dengan fungsi atau kelas view yang sesuai. Dengan memahami sistem routing, kita dapat menentukan bagaimana aplikasi merespons berbagai permintaan pengguna berdasarkan pola URL yang diakses.
