# 06: Functional Testing with WebTest

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami **Functional Testing** pada aplikasi Pyramid menggunakan **WebTest**, yaitu metode pengujian yang mensimulasikan interaksi pengguna dengan aplikasi web secara utuh.  
Berbeda dengan Unit Test yang menguji fungsi individu, **Functional Test** menguji seluruh proses dari request ke response seperti yang terjadi pada browser nyata.

## Analisis
Functional Testing bertujuan untuk memastikan bahwa alur kerja (workflow) dari aplikasi berjalan sesuai dengan fungsionalitas yang diharapkan. Sedangkan WebTest library Python yang digunakan untuk menguji aplikasi web WSGI seperti Pyramid tanpa harus menjalankan server.
