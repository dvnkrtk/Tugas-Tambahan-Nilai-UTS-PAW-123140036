# 07: Basic Web Handling With Views

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami cara dasar menangani permintaan web (HTTP Request) dan menghasilkan respons (HTTP Response) menggunakan View pada framework Pyramid.

## Analisa
View di Pyramid adalah komponen utama yang menerima permintaan dari pengguna (request) dan mengembalikan hasil (response). View biasanya berupa fungsi atau class yang dikaitkan dengan route tertentu. Berdasarkan analisa saya, pada guide ini diberitahu:
- Membuat fungsi view.
- Menghubungkan view dengan route.
- Mengatur response baik berupa teks maupun JSON.
- Menjalankan aplikasi Pyramid dan mengakses hasilnya di browser.
