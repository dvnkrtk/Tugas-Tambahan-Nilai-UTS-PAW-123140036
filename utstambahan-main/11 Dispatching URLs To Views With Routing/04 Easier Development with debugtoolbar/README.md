# 04: Easier Development with debugtoolbar

## Tujuan Percobaan
Percobaan ini bertujuan untuk mempermudah proses pengembangan aplikasi Pyramid dengan menggunakan Pyramid Debug Toolbar. Fitur ini membantu pengembang menganalisis request, response, routing, template rendering, SQL query, serta error secara interaktif langsung dari browser.

## Analisa
Berdasarkan analisa saya, pada guide ini diberitahu cara menginstall debugtoolbar supaya memudahkan kita melakukan pengembangan web karena errornya lebih mudah ditracking menggunakan alat ini. Hanya boleh diaktifkan pada mode development (`development.ini`), tidak untuk production karena dapat mengungkapkan konfigurasi internal aplikasi.
