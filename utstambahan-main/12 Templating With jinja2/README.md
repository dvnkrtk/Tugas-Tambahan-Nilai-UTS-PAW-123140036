# 12: Templating With Jinja2

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami bagaimana menggunakan **Jinja2** sebagai sistem templating di dalam framework Pyramid. Templating berfungsi untuk memisahkan logika aplikasi dari tampilan, sehingga kode menjadi lebih terorganisir, mudah dibaca, dan dapat dikelola oleh pengembang maupun desainer secara terpisah.

## Analisis
Berdasarkan hasil percobaan, dapat dipahami bahwa Jinja2 adalah sistem templating yang memungkinkan pemisahan antara tampilan dan logika. Fungsi view hanya perlu mengembalikan data dalam bentuk dictionary, sedangkan template bertugas untuk menampilkannya dalam bentuk HTML.
