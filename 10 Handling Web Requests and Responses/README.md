# 10: Handling Web Requests and Responses

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami bagaimana Pyramid menangani HTTP Request dan HTTP Response di dalam sebuah aplikasi web. Dalam percobaan ini, kita mempelajari bagaimana data dari permintaan (request) seperti parameter URL, form, dan header dapat diakses oleh view, serta bagaimana membentuk **respon (response)** yang sesuai untuk dikirim kembali ke browser.

## Analisis
Berdasarkan analisis saya, pada percobaan ini dijelaskan bagaimana Pyramid menyediakan objek `request` dan `response` yang digunakan untuk berinteraksi dengan data web.
