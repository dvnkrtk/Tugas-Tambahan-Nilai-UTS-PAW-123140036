# 19: Databases Using SQLAlchemy

## Tujuan Percobaan
Percobaan ini bertujuan untuk memahami bagaimana mengelola database pada aplikasi web Pyramid menggunakan SQLAlchemy. SQLAlchemy digunakan untuk mempermudah interaksi dengan database melalui ORM (Object Relational Mapping) sehingga developer dapat melakukan operasi database menggunakan objek Python tanpa menulis query SQL secara langsung.

## Analisis
Berdasarkan hasil percobaan, SQLAlchemy memungkinkan pengelolaan database menjadi lebih efisien dan terstruktur. Pembuatan tabel, penyimpanan data, pembaruan, penghapusan, dan query dapat dilakukan melalui model Python. Selain itu, SQLAlchemy memudahkan pengaturan relasi antar tabel dan memastikan integritas data melalui constraint dan relationship. Dengan menggunakan SQLAlchemy, pengembang dapat mengurangi kesalahan dalam query, meningkatkan keterbacaan kode, dan mempercepat proses pengembangan aplikasi web berbasis database.
